
NodeJS- это платформа (среда выполнения) на основе движка V8, которая позволяет запускать JavaScript вне браузера. 

Глобально она переводит JavaScript код в машинный код.

## **Столпы NodeJS**

###  ## Движок V8 

движок от гугл, написанный на C++, который занимается трансляцией JavaScript кода в машинный код. Грубо говоря его задачей является компиляция и выполнения кода на Javascript. Также на его плечах лежит выделение памяти под объекты и сборка мусора.

#### Основные функции:
1.  Компиляция и выполнения кода на Javascript
2. Выделение памяти
3. Сборка мусора

## Libuv

Библиотека, написанная на С, лежащая в основе NodeJs она отвечает за:

1. Асинхронный и неблокирующий ввод-вывод (I/O)
2. Управление циклом событий (Event Loop)
3. Работа с сокетами, TCP/UDP,DNS, файловыми системами, потоками, таймерами и тд.

**libuv превращает однопоточный JavaScript в высокопроизводительную, асинхронную платформу** за счёт работы на низком уровне.
